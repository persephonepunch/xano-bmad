## üèóÔ∏è Tech Stack

**Frontend:** Eleventy (11ty), Webflow designs (NO Tailwind)

**Hosting:** Netlify (static + serverless)
**Auth:** Auth0 + permit.io (ABAC policies)
**Payments:** Stripe + Plaid (ACH 0.8% fees)
**CDN:** Bunny.net (cost-optimized)
